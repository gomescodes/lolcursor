## Test Locally

Clone the project

```bash
  git clone https://github.com/gomescodes/lolcursor
```

Go to the project directory

```bash
  cd lolcursor
```

Install dependency

```bash
  pip install PySide6
```

Start the app

```bash
  python lolcursor.py
```

## Building Executable

Install dependency

```bash
  pip install pyinstaller
```

Run pyinstaller

```bash
  pyinstaller .\lolcursor.py -F -w --icon=favicon.ico --add-data="favicon.ico;."
```
